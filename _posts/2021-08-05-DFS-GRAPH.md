#### 一、leetcode： [802. 找到最终的安全状态](https://leetcode-cn.com/problems/find-eventual-safe-states/)
```
# 深度搜索/递归 + 标记
class Solution:
    def eventualSafeNodes(self, graph: List[List[int]]) -> List[int]:
        n = len(graph)
        color = [0]*n

        def safe(x: int) -> bool:
            if color[x] > 0:
                return color[x] == 2
            color[x] = 1
            for y in graph[x]:
                if not safe(y):
                    return False
            color[x] = 2
            return True
        return [i for i in range(n) if safe(i)]
```
