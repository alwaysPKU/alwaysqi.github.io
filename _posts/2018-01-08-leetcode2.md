### 题目

题目的意思是给两个链表，分别代表两个数

```
You are given two non-empty linked lists representing two non-negative integers. The digits are stored in reverse order and each of their nodes contain a single digit. Add the two numbers and return it as a linked list.

You may assume the two numbers do not contain any leading zero, except the number 0 itself.

Example:

Input: (2 -> 4 -> 3) + (5 -> 6 -> 4)
Output: 7 -> 0 -> 8
Explanation: 342 + 465 = 807.


```

求返回的链表。

### 方案

主要就是考虑到是否会进位的问题，两个的长度不一定一样，比如[2,3]  和[3,4,5]， 但是返回的链表肯定是最长的那个的长度，或者还要长，因为可能进位。所以继续进行下去的条件是或者两个数还没有用完（至少有一个没有用完），或者要进位

```
class Solution:
    def addTwoNumbers(self, l1, l2):
        carry = 0
        ret = ListNode(0)   # 初始化为0
        root = ret  # 因为ret会不断地更新
        while l1 or l2 or carry:   # 这三个条件只要有一个满足就应该继续下去
            v1 = v2 = 0
            if l1:
                v1 = l1.val
                l1 = l1.next
            if l2:
                v2 = l2.val
                l2 = l2.next
            carry, val = divmod(v1+v2+carry, 10)   # 这个函数很好用
            
            ret.next = ListNode(val)  # 第一个0的没有要
            ret = ret.next   ## 别忘了更新ret
        
        return root.next   # 因为第一个0不是的，
            
# 注意不要返回成 ret.next了，这样会错，因为ret一直在更新。

```
