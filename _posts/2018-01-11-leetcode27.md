### 题目

这个题目和26题很相似，也可以从后面开始，不过还可以用erase的返回值来不断地更新

```

class Solution {
public:
    int removeElement(vector<int>& nums, int val) {
        vector<int>::iterator iter = nums.begin();
        while(iter != nums.end()){
            if(*iter == val) iter = nums.erase(iter);
            else ++iter;
        }
        return nums.size();
    }
};

```

其实之前的26题，也可以用erase的这种办法。
