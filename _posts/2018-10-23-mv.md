mv 命令非常方便，移动，和重命名，但是有时候要注意，

比如下面的操作

```
mkdir HHHH

cd HHHH
touch 1.txt
touch 2.txt

cd ..

mv HHHH/ HHH

```

上面操作的时候如果，'HHH'这个文件夹是没有的话，那么结果会导致，'HHH' 和'HHHH'这两个都会消失不见，所以操作的时候一定要注意
正确的操作是

```
mv HHHH HHH

```
就会对上述文件夹重新命名。
